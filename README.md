# Estudos direcionados ao uso de SQL para Análise de Dados

Este repositório tem como finalidade conter e apresentar exercícios e consultas SQL desenvolvidos durantes meus estudos e evolução no uso de da linguagem SQL para análise de dados.

# Conteúdos envolvidos
- SELECT e WHERE
- JOINs
- Queries e SubQueries
- Funções de string e cálculos numéricos
- Agregações e contagens (COUNT, GROUP BY)

# Ferramentas
- MySQL
- VSCODE
- Sqlite Explorer
- MySQL Workbench

# Fontes de informações

Gostaria de prestar as saudações aos livros educativos da O'Reilly pela base teórica de SQL para estudos, sendo eles:
- SQL Guia Prático - Um guia para o uso de SQL
- Introdução ao SQL
